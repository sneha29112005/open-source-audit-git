# Open Source Software Audit — Git

## 📌 Overview

This project is a structured audit of Git, a distributed version control system created by Linus Torvalds.

It explores:

* Origin and philosophy of open source
* GPL licensing
* Linux integration
* FOSS ecosystem
* Comparison with proprietary tools

---

## 🧠 Why Git?

Git represents the true spirit of open source:

* Built to solve a real problem
* Shared freely
* Maintained by a global community

---

## 🖥️ Scripts Included

1. System Identity Report
2. Package Inspector

4. Log Analyzer
5. Manifesto Generator
